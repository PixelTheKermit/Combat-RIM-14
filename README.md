## Contributing

Contributions are not guaranteed to be accepted. Anything larger than a bugfix should be discussed in Discord beforehand, or your PR will immediately be closed. Contributions should be sent upstream whenever possible, and this will be enforced.

## Building

1. Clone this repo.
2. Run `RUN_THIS.py` to init submodules and download the engine.
3. Compile the solution.

[More detailed instructions on building the project.](https://docs.spacestation14.io/getting-started/dev-setup)
